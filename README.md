## Getting Started

Welcome:

First, install the dependencies:

```bash
yarn 
```

Second, run the development server:

```bash
yarn dev
```

Go to [http://localhost:3000/api/post](http://localhost:3000/api/post) and use the following queries to call the API:
    name        type
    tags        string
    sortBy      string
    direction   string
